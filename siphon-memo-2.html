Siphon memo 2

Siphon は一度 view map を計算さえすれば、feature edges から stroke の生成はごく短時間で可能であるという予想で開発を進めている。そのためには view map を bake する必要がある。とりあえず以下のコマンドで RegionView3D をロックすることができる。

context.space_data.region_3d.lock_rotation = True

Siphon の NodeTree 編集操作を hook しておき Render Result をリアルタイムに更新するという方法も考えている。
